# pet-management
This is a group project focused on building a pet management system for an animal shelter.

The goal is to make the adoption process easier by allowing users to view pets available for adoption and give up their own pets for rehoming.

## technologies used
- HTML/CSS/JS and bootstrap
- ASP.NET core (.NET 8.0)
- Entity Framework (turning our classes into SQL tables)
- SQL Database
- Microsoft Azure (cloud server for our Database and blob for saving the pictures)

## How to run the project
- fetch the latest version of main
- make sure you have the correct .NET version 
- run the project using rider or visual studio
- in case of any errors first make sure the CORS in Program.cs is right (mostly happenes to mac users)
- in case of error not being resolved, can go and use the local-version branch instead

## Project Structure
- all the code related files are in the backend folder
- the query file export of our databse can be fine in databse.sql (you can run this file in your local database to see the darabse with some data).
- all the daily meetings, MOSCOW tables and the report can be find in the documents folder
